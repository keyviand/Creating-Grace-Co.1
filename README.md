# KIT Creations Web App

This is a starter web application for selling handmade sewing creations and advertising custom IT app services.

## Files
- `index.html` - Main website page
- `styles.css` - Design and mobile layout
- `app.js` - Product cards, menu, and contact form
- `manifest.json` - Lets the website be added to a phone home screen
- `service-worker.js` - Basic offline caching
- `logo.svg` - App/logo icon

## What to edit first
1. Open `index.html` and change:
   - Business name
   - Contact email
   - Phone number
   - Location
2. Open `app.js` and change:
   - Product names
   - Prices
   - Descriptions
   - `yourname@example.com` to your real email
3. Replace `logo.svg` if you want a different logo.

## How to upload to GitHub Pages
1. Create a GitHub repository.
2. Upload all files in this folder.
3. Go to Settings > Pages.
4. Under Branch, select `main` and `/root`.
5. Save.
6. GitHub will give you a public website link.

## Home screen app logo
After publishing the site:
- iPhone: Open in Safari > Share > Add to Home Screen.
- Android: Open in Chrome > Menu > Add to Home screen.
